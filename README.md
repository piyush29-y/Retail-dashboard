# Retail Company Sales and Customer Demographic Dashboard

This project is an interactive Power BI dashboard developed to analyze retail company sales performance and customer purchasing behavior. The dashboard provides valuable insights into product sales, revenue generation, order distribution, and customer demographics through visually appealing charts and KPIs.

## Features

- Total Orders and Revenue KPIs
- Category-wise product quantity analysis
- Product-wise quantity and sales analysis
- Total price contribution by each product
- Daily order quantity trends
- City-wise order distribution
- Interactive and user-friendly dashboard design

## Dashboard Insights

- Electronics products contribute the highest sales quantity.
- Mouse and Laptop products generate the highest revenue.
- Mumbai has the highest number of orders among all cities.
- Daily order analysis helps identify customer purchasing patterns.

## Technologies Used

- Power BI
- Data Visualization
- Business Intelligence
- Retail Sales Dataset

## Project Objective

The objective of this project is to help retail businesses monitor sales performance, understand customer buying patterns, analyze product demand, and support data-driven business decisions through interactive visualizations.
